# 河源电视直播订阅

给广东河源地区电视或 IPTV 播放器使用的精简 M3U 订阅，每天自动从
[`fanmingming/live`](https://github.com/fanmingming/live) 更新。

## 电视订阅链接

```text
https://icarsone.github.io/heyuan-tv/heyuan.m3u
```

频道按兼容性分成三组：

- `广东联通优先`：广东联通宽带优先尝试。
- `IPv6备用`：光猫、路由器和电视已开启 IPv6 时使用。
- `通用备用`：前两组不能播放时尝试。

直播源来自互联网，仅供测试研究使用，可能随时失效。仓库只整理订阅文件，不存储或转播任何视频内容。
